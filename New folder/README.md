# MegaMart

A lightweight static storefront demo built with HTML, CSS, and JavaScript.

## Features

- Product catalog with images, categories, pricing, ratings, and descriptions
- Search products by name
- Category filtering
- Rating filtering
- Price filtering with slider and quick preset buttons
- Sorting by price and rating
- Pagination for product listing
- Wishlist toggle on product cards
- Shopping cart with item count, remove item, and checkout flow
- Product detail modal with add-to-cart support
- Dark mode toggle with persistence
- Local storage persistence for cart, wishlist, and theme
- Toast notifications and click sound effects for interactions

## Files

- `index.html` — page structure and modals
- `style.css` — responsive visual design and dark mode styles
- `script.js` — product data, filtering, cart, wishlist, modal behavior, and persistence

## Usage

1. Open `index.html` in a browser.
2. Use the search bar, filters, and sort menu to explore products.
3. Add items to your cart or wishlist.
4. Open the cart to review items and complete checkout.
5. Toggle dark mode with the moon/sun button.

## Notes

- This project is entirely client-side and does not require a server.
- Cart, wishlist, and theme settings are stored in browser `localStorage`.
