# MegaMart E-Commerce Storefront

## Overview
MegaMart is a frontend e-commerce storefront built using HTML, CSS, and JavaScript.

The project allows users to search products, apply filters, sort products, manage a shopping cart, and maintain a wishlist.

## Features

### Required Features
- Search by product name
- Category filter
- Price filter
- Rating filter
- Product count display

### Additional Features
- Product sorting
- Pagination
- Shopping cart
- Wishlist
- Product detail modal
- Dark mode
- Toast notifications
- LocalStorage persistence
- Image fallback handling
- Responsive design

## Technologies Used
- HTML5
- CSS3
- JavaScript
- ChatGPT
- GitHub Copilot

## How to Run
1. Download or clone the repository.
2. Open the project folder.
3. Open index.html in a browser.

No installation is required.
